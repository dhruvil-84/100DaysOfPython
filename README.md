# 100 Days, 100 Projects in Python 🐍

## Overview
Welcome to the **100 Days, 100 Projects in Python** repository! 🎉  
This repo contains a collection of Python projects completed as part of the Udemy course "100 Days of Code: Python Developer Challenge".

Each day presents a new project designed to sharpen your Python skills and cover a wide variety of programming topics. From beginner to advanced, the projects provide a hands-on approach to learning Python's functionality, libraries, and real-world application development.

## Course Goals 🏆
- Build confidence with Python syntax, libraries, and best practices.
- Gain experience with a variety of Python projects, including web development, automation, data analysis, and more.
- Learn to implement object-oriented programming, algorithms, and data structures.
- Enhance problem-solving abilities by tackling different challenges daily.

## Table of Contents 📚
Below is a brief description of each project. Explore the folders for more detailed code and explanations:

1. **Day 1-10: Python Basics**  
   - Variables, Data Types, Control Flow, Functions, Loops, and Basic I/O.
   
2. **Day 11-20: Intermediate Projects**  
   - File Handling, Error Handling, Modules, and more advanced functions.
   
3. **Day 21-30: OOP (Object-Oriented Programming)**  
   - Classes, Objects, Inheritance, Polymorphism, and Encapsulation.
   
4. **Day 31-40: GUI Development**  
   - Using `Tkinter` to create simple graphical user interfaces.

5. **Day 41-50: Web Scraping**  
   - Projects involving scraping data from websites using `BeautifulSoup` and `Selenium`.

6. **Day 51-60: APIs & Networking**  
   - Working with REST APIs, HTTP requests, and interacting with web services.

7. **Day 61-70: Data Science and Visualization**  
   - Data Analysis, Pandas, Matplotlib, and Seaborn projects.

8. **Day 71-80: Automation & Scripting**  
   - Writing automation scripts for file handling, web tasks, and data processing.

9. **Day 81-90: Web Development with Flask/Django**  
   - Building simple web applications using Flask and Django.

10. **Day 91-100: Final Projects**  
    - Larger, more complex projects to wrap up the course.
